A Date and Time Picker plugin for jQueryMobile

Flip Default

Using the flip mode is as simple as setting the 'mode' option to "flipbox".

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "flipbox"}'>

Flip Time Mode Default

Using the flip mode is as simple as setting the 'mode' option to "timeflipbox".

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "timeflipbox"}'>

Flip Time Mode 12 Hour Clock

Using the flip mode is as simple as setting the 'mode' option to "timeflipbox".

<label for="mydate">Some Date</label>

<input name="mydate" id="mydate" type="date" data-role="datebox"
   data-options='{"mode": "timeflipbox", "timeFormatOverride": 12}'>
